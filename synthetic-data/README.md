# Synthetic Data Generators

> **Status:** In Entwicklung. Erste Testdaten befinden sich in [`labs/lab-opa-sandbox/init-db.sql`](../labs/lab-opa-sandbox/init-db.sql).

Dieses Verzeichnis wird Python-basierte Generatoren für synthetische Testdaten enthalten, die für Compliance-Übungen verwendet werden können – ohne echte Personendaten (DSGVO-konform).

## Geplant

- **Transaction Generator**: Synthetische Finanztransaktionen mit eingestreuten AML-Mustern
- **Identity Generator**: Fiktive KYC-Profile mit verschiedenen Risikostufen
- **Incident Generator**: ICT-Incidents mit verschiedenen Schweregrad-Kombinationen
